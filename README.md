# Manga Bot

Manga Bot is a user-friendly and efficient application that can help you quickly grab your favorite manga series, enabling you to save it to your computer and read it whenever you want.

It's **100% FREE** without any ads in app. If you have any inquiries or difficulties in use, just feel free to feedback to me via email [dangduy2910@gmail.com](mailto:dangduy2910@gmail.com) or leave comments here.

## Features

- Download up to 10 manga simultaneously with one-click (unlimited in Queue)
- Automatically update manga list. No longer worry about the manga list is out of dated
- Download from any link (Grabber tool)
- Create ZIP/PDF files automatically after downloaded
- Advanced search across sites
- Shutdown your computer after download queue was completed
- Create ZIP/PDF file from anything downloaded manually (Converter tool)

## FAQs

1. How to download the manga immediately?

> Choose manga/chapters you want to download (on the left/right list) -> right-click on the list -> Download.
> Remember that you can only download up to 10 manga at the same time.

2. How to bookmark and download manga later or more than 10 manga at the same time?

> Choose manga/chapters you want to download (on the left/right list) -> right-click on the list -> Add to Queue. It should be shown in the bottom list.
> Remember that it just added to the list. To start downloading, click button Start Queue ![SQ](assets/multidownload.png).
